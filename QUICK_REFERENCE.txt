🐛 COMMON ISSUES & SOLUTIONS
──────────────────────────────────────────────────────────────────

Issue: npm run test:cloudkilat fails
Fix  : 1. Check CLOUDKILAT_SECRET_KEY in .env
       2. Make sure bucket exists in CloudKilat

Issue: Database connection error
Fix  : 1. Check PostgreSQL is running
       2. Verify DATABASE_URL in .env
       3. Test: psql -U postgres -d kilatbox

Issue: Port 3000 already in use
Fix  : 1. Edit .env, change PORT=3001
       2. Restart server

Issue: Upload fails
Fix  : 1. Check bucket permissions
       2. Verify credentials
       3. Run: npm run test:cloudkilat
